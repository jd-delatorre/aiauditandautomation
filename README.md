# NJ AI Automation

Business materials for providing AI Automation Audits and implementation services (powered by OpenClaw) to NJ businesses.

## Goal
Generate $5,000/month through audit fees, implementation projects, and monthly retainers.

## Contents

- **strategy.md** - Full business strategy, audit model, use cases, pricing, and market research
- **tasks.md** - Implementation checklist organized by phases
- **landing-page/** - Marketing website (deploy to Vercel)
- **landing-page-content.md** - Landing page copy and FAQ content
- **video-script.md** - Sales/demo video script

## Quick Start

### Deploy Landing Page
```bash
cd landing-page
npx vercel
```

### Customize
1. Update email in `landing-page/index.html`
2. Add your Calendly link (search for `YOUR_CALENDLY_LINK`)
3. Register domain (njaiautomation.com)

## Service Flow

1. **AI Automation Audit** — Analyze business workflows, deliver report with 5-15 opportunities
2. **Recommendations** — Prioritized roadmap with ROI estimates
3. **Implementation** — Build automations using OpenClaw
4. **Ongoing Support** — Monthly retainers for maintenance and expansion
